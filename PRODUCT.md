### 2. Research Is Cheaper Than Wrong Data
In any domain where accuracy matters, one wrong data point categorizes the tool as "unreliable" and the user stops trusting it. The cost of agent research (minutes, pennies) is negligible compared to the cost of showing wrong data (lost user, lost trust).

**When this applies:**
- Before shipping any new data, verify against primary sources
- Before building a feature that displays data, research ALL supported items
- When choosing between "ship fast with assumptions" and "research first then ship," research first

**Anti-pattern:** "We'll research that later when we have time."
